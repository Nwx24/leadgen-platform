# LeadGen Platform

A lightweight full-stack lead capture and management web application built with Flask.  
The platform allows businesses to collect, validate, and store customer inquiries through customizable web forms and provides a foundation for building industry-specific lead generation funnels (real estate, local services, consulting, and more).

- Responsive lead capture form
- Server-side validation for reliable data collection

- Contractor quote requests
- Local service booking leads
- Small business marketing funnels
