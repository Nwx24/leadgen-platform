# Real Estate Lead Generator (Flask)

A lightweight full-stack web application that captures real estate leads through a web form and stores them locally.

## Features
- Lead capture form (name, email, phone, intent, timeline, message)
- Server-side validation
- CSV-based lead storage
- Thank-you redirect after submission

## Tech Stack
- Python
- Flask
- HTML / CSS

## How to Run
1. Create a virtual environment
2. Install Flask
3. Run:
   python app.py
4. Open the local server URL
